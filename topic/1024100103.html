HitChain Project Biweekly Report：08 October, 2018—19 October, 2018

October 19, 2018

1. Project Progress

1) Realized HIT web prototype and service interface

a、User regist and security control.

b、 Create and migrate the project.

c、Digital developers authorization management.

d、Webservice of HIT protocol.

d-1、 Design and realized getIpfsHash and changeIpfsHash interface. Finished version control and network domain mapping features.

d-2、 Design and realized hitTransfer interface and provides user authentication and project uniqueness detection required for project transfer from the collaborative development community to the IPFS network.

2) Update hit protocol.

a、Migration projects across the different communities: migrate projects from other platforms with one operation. You can migrate the project to your owner by provided username and password and project name.

b、Reform the hit remote address: abandoned immature ipns features and realized hit address and ipfs hash address mapping directly. Users can analysis the ipfs hash by hit web service.

c、Upgrade and optimize the native git functions:

c-1、hit push hit_url: to add authorization control.

c-2、hit pull: auto update remote address.

c-3、hit clone: use the project name on the platform recorded when the project name is empty and not use the folder name any more.

c-4、hit commit & add: improve the efficiency of the distributed multi-level commit.

c-5、hit transfer: extend migrate projects cross the community features can migrate from any community.

3) Finished building development environment.

a、Finished building Gitlab development environment and testing including PR commit.

b、Finished building Gitaly and Gitlab development environment and testing.

c、Submitted artifacts: https://github.com/hitchainrepo/hitlab/tree/master/doc/hitlab/dev, 01SetupGdkDevEnvironment.md, 02SetupDebugEnvironment.md, 03SetupRemoteGitaly.md

4) Research on alternatives to Gitlab

a、Doing research of Gitbucket includes features and development convenience.

b、Doing research of Gitblit includes features.
