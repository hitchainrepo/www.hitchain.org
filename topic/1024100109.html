HitChain Project Biweekly Report:2 January—13 January, 2019

January 13, 2018

1. Project Progress

1) From January 6th to 7th, HitChain was invited to participate in the year-end ceremony of GBLS Global Sleepless Blockchain Industry Leaders. Chen Zhao, senior architect of HitChain, and the technical team participated in the conference, mainly discussed with the IPFS-related project parties on decentralized content storage. Chen Zhao said that HitChain needs to learn from IPFS in code decentralized storage and hosting. Cooperation with IPFS will greatly promote the technical and ecological development of both parties.

2.Project Progress

1) Changes for the server’s IPFS:

a、Do not need to add parameters when initializing, and automatically add the existed server nodes into the peer list.

b、Transplant the server’s listener into IPFS.

c、Automatically monitor the storage capacity when initializing, and allocate 80% of the remained storage as the IPFS repository storage. When the remained storage is less than 50GB, warning.

2) Changes for the client’s IPFS:

a、Visiting the user selected server for node reporting when initializing.

b、When updating the repository, asking the selected server for backup automatically.

3) Update architecture design to change the IndexFile definition and add GitFileIndex file definition.

4) Integrate the Ethereum technology to record the project address.

3.Market Dynamics

1)Due to the strategic adjustment of the HitChain Foundation, after careful consideration and friendly negotiation with the OKEx team, HitChain (HIT) delisted OKEx. OKEx officially delisted HIT all trading pairs at 19:00 on January 11, 2019, Hong Kong time. All HIT users can continue to trade on the trading platforms of Huobi Global, Gate.io, and CoinW.

8) Add hash record feature to Hitbucket to record the repository hash value. This feature will change to publish the hash to the ethereum.
