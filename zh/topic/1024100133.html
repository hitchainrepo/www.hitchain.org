HitChain 项目双周报（2019年第21期）：10月7日～10月20日

2019-10-20

一、技术研发

1）升级 hit1.0-specs 配置：

a）更新 ProjectInfoFile 的配置项，移除 ethereumUrl 配置，以太坊智能合约的入口由 hit config 配置；

b）更新 hit config 的配置项，移除 repository 配置，该配置为旧的加速器配置，不再使用；

c）更新 hit config 的配置项，更新 storage 配置内容，允许 IPFS 的自定义端口配置；

d）更新相关的代码，移除 ethereumUrl、repository 等配置代码，更新 storage 配置代码；

e）更新 ProjectInfoFile 测试代码；

2）双 POS 复合主链研究设计：

a）Proof of Stake 帐本主链研究设计；

b）Proof of Storage 存储主链研究设计;

c）研究整合 Proof of Stake 及 Proof of Storage 两种主链技术及零知识证明技术，为主链提供去中心化帐本及去中心化存储功能；

二、研发计划

1）9月份计划

a）实现 PullRequest 的代码加密；

b）实现代码仓库浏览器，可检索区块链上的代码仓库项目信息；

2）后续计划

a）实现 Pull Request 浏览器，支持可视化 Pull Request 浏览；

b）实现 hit 钱包，用于管理所属的仓库、 HIT 代币交易、ETH 代币交易。
