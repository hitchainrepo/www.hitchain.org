HitChain 项目双周报（2019年第20期）：9月23日～10月6日

2019-10-6

一、技术研发

1）新增仓库 hit1.0-specs：

a）新增 hit1.0-specs 仓库，用于存储所有规范的文档；

b）整理 Project Information File 规范内容，详细说明该文档的结构及使用方法；

c）整理 Git File Index 规范内容，详细说明该文档的结构及使用方法；

d）仓库位置：https://github.com/hitchainrepo/hit1.0-specs.git。

2）仓库浏览器开发：

a）新建项目信息详情页面，浏览项目的创建者、团员成员等基本信息；

b）仓库浏览器中以太坊区块浏览服务由 infura web3j 切换到 etherscan api，解决在中国区网络很可能无法访问 infura 服务的问题;

二、研发计划

1）9月份计划

a）实现 PullRequest 的代码加密；

b）实现代码仓库浏览器，可检索区块链上的代码仓库项目信息；

2）后续计划

a）实现 Pull Request 浏览器，支持可视化 Pull Request 浏览；

b）实现 hit 钱包，用于管理所属的仓库、 HIT 代币交易、ETH 代币交易。
